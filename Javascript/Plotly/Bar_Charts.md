# Use Case
## Adjust the y-axis base
### Sample Code
```
const data = [
 {
    x: ['giraffes', 'orangutans', 'monkeys'],
    y: [20, 14, 23],
    base:[10,-10], // the default offset is 0
    type: 'bar'
  }
];
```
