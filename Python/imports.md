# Overview

| JS | PY |
|---|--- |
| ```import * as "name" from "module_name"``` | ```import "module_name" as "name" ```
| ```import {"sub_module"} from "module_name"``` | ```from "module_name" import "sub_module"```

# Standard Library

| Language | Example |
|---|---
| JS | Math.sqrt(64)
| PY | import math<br>...<br> math.sqrt(64)
